# Nomor 3
3 Looping yang dapat digunakan yaitu FOR, WHILE, dan DO WHILE
- FOR berfungsi untuk melakukan perulangan jika sudah tau ingin mengulangi berapa kali
- WHILE digunakan jika tidak mengetahui berapa kali merlakukan perulangan
- DO WHILE digunakan ketika kita ingin setidaknya menjalankan pengulangan 1 kali sebelum dilakukan pengecekan kondisi
